# Claude Code Workshops

Hands-on workshops for learning Claude Code, from beginner fundamentals to advanced features.

## Getting Started

### Prerequisites

- Node.js installed
- Claude Code installed:
  ```bash
  npm install -g @anthropic-ai/claude-code
  ```
- Basic command line familiarity

### Installation

```bash
git clone https://github.com/gravity9-tech/claude_code_workshop_test.git workshops
cd workshops
```

## Workshops

| Workshop | Level | Duration | Description |
|----------|-------|----------|-------------|
| [Workshop 1](./cc-workshop-1/) | Foundation | ~45 min | Context window, MCP, slash commands, subagents |
| [Workshop 2](./cc-workshop-2/) | Advanced | ~58 min | Skills, hooks, marketplace, plugins, workflow automation |
